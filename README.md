## Sobre este repositório

Este repositório reúne projetos e testes criados no início dos meus estudos em programação,
antes mesmo de eu aprender versionamento com Git e boas práticas de commits.

O objetivo é registrar minha evolução técnica ao longo do tempo.

Projetos mais recentes e com melhor organização podem ser encontrados nos meus outros repositórios ou fixados no perfil.
